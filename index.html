Welcome to TrackmaniaStats

Disclaimer!

This website only exist because I found it cool to be able to see everyones performance on the Cup of the Day, a knot-out daily competition hosted by Nadeo.

You can check you own profile or the ones of the others, so please, do not use this tool to judge / make fun of somebody or anything, because some players play the cotd just for fun (so no judgement allowed here).

To access a profile, enter a name (or just a part of a name), and you will see all corresponding player that match your search. To see someone's profile, click on the see profile button
